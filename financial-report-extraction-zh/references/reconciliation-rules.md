## 请对用户提供的各类格式客户资产负债表进行科目平衡性校验,执行规则如下：

- 自动识别报表内全部资产类合计、负债类合计、股东权益合计.....
- 计算资产负债表中相关勾稽关系是否平衡
- 识别漏填合计数、合计数计算错误、科目错放、期末期初数混淆等常见填报问题
- 最终输出清晰结论：报表是否平衡、差额多少、存在哪些填报漏洞、简要整改方向

## 核心校验规则（必须严格执行，不可违反）

- 【公式1、公式2】校验：资产总计 = 负债和所有者权益总计（直接使用报表合计数对比）
  - 公式1：资产总计（期初）= 负债和所有者权益总计（期初）
  - 公式2：资产总计（期末）= 负债和所有者权益总计（期末）
- 【公式3、公式4】校验：资产总计 = 全部【末级明细资产科目】相加（禁止使用任何合计科目；计算时，如果存在【原值 / 原价】，禁止取【净额】，必须用【原值 / 原价】- 【折价等】计算，若没有【原值 / 原价】，则可以取净值）
  - 公式3：资产总计（期初）= 所有末级资产明细科目相加（期初）
  - 公式4：资产总计（期末）= 所有末级资产明细科目相加（期末）
- 【公式5、公式6】校验：负债和所有者权益总计 = 全部【末级负债明细科目】+ 全部【末级权益明细科目】相加（禁止使用任何合计科目，必须用明细）
  - 公式5：负债和所有者权益总计（期初）= 所有末级负债明细 + 所有末级权益明细（期初）
  - 公式6：负债和所有者权益总计（期末）= 所有末级负债明细 + 所有末级权益明细（期末）
- 如果没有期初值则全部变量按照0处理
- 公式3/4/5/6 绝对不允许使用：流动资产合计、固定资产合计、负债合计、权益合计等包括小的合计项目任计算，要使用子项进行计算，
- 区分取值期间仅在「期间」字段填写，**科目名称本身绝对不能携带期间标识**
- 计算时，如果存在【原值 / 原价】，禁止取【净额】，必须用【原值 / 原价】进行计算
- 计算时，如果不存在【原值 / 原价】的时候可以取净值
- 所有出现的科目名称必须完整，禁止出现丢失前面的序号，丢失括号内容等情况

## 请对用户提供的各类格式客户利润表进行科目平衡性校验,执行规则如下: 

- "本月值","本年累计" 分别是使用一个公式进行整体校验  
- 计算利润表中相关勾稽关系是否平衡  
- 识别漏填合计数、合计数计算错误、科目错放、期末期初数混淆等常见填报问题  
- 最终输出清晰结论：报表是否平衡、差额多少、存在哪些填报漏洞、简要整改方向  
  
## 注意事项  

- 从头开始逐项按照逻辑计算以最后一个科目（净利润/归属于母公司所有者的净利润/（或者除了前两个其他在底部的科目）...）为结果科目,最后验证勾稽计算结构是否与表中数据一致  
- 【公式情况】中只有两项,分别计算"本月值","本年累计"  
- 所有出现的科目名称必须完整，禁止出现丢失前面的序号，丢失括号内容等情况  
- 如果没有本月值，则全部置为0  
- 如果有名为【资产减值损失】或名字包含【资产减值损失】的科目，无论该科目值是正数还是负数，计算时都必须使用加法进行计算
